# Todo-List App

A simple and responsive Todo List application built with vanilla HTML, CSS, and JavaScript with data persistence enabled via LocalStorage.

## Features

- **Add Tasks**: Add new tasks to the list. Input validation ensures no empty tasks are added.
- **Mark as Completed**: Click on a task to toggle its completion status (strikethrough effect).
- **Delete Tasks**: Remove tasks from the list permanently using the delete button.
- **Data Persistence**: Tasks are saved in the browser's LocalStorage, so they remain available after refreshing the page.

## Project Structure

- `index.html`: The main structure of the application.
- `style.css`: Styles for the application
- `script.js`: Logic for handling task addition, deletion, completion toggling, and LocalStorage updates.
