# Image Gallery Lightbox


## Features

- **Responsive Gallery**: Displays images in a flexbox layout that wraps automatically.
- **Lightbox Modal**: Click on any image to view it in a full-screen modal overlay.
- **Interactive Controls**:
  - Click logic handles opening the modal with the correct image.
  - Close button (`×`) to exit the view.
  - click on the backdrop (outside the image) to close the modal.
  - Press `Escape` key to close the modal.
- **Hover Effects**: Images have a scaling effect on hover.
